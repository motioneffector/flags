# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-11

### Added
- Initial release
- Flag store supporting boolean, number, and string values
- Powerful condition evaluation with logical operators (AND, OR, NOT) and comparisons
- Support for numeric comparisons (>, <, >=, <=, ==, !=)
- String equality checks with proper escaping
- Change subscriptions (global and per-key)
- Storage persistence with configurable backends
- History system with undo/redo support
- Namespacing for organized flag management
- Batch updates for atomic operations
- Computed flags for derived values
- Full TypeScript support with complete type definitions
- Zero dependencies
- Comprehensive test suite (528 tests)
