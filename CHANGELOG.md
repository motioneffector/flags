# Changelog

## [0.0.1] - 2026-01-11

### Added
- Initial implementation
- Flag store with boolean, number, and string values
- Condition evaluation with logical operators (AND, OR, NOT) and comparisons
- Change subscriptions (global and per-key)
- Persistence to storage backends
- History with undo/redo support
- Namespacing for organized flag management
- Batch updates for atomic operations
- Computed flags for derived values
